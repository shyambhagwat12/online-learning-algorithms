
# ETC Algorithm
Features:
1. Simple uncontrained alg.
2. For a known suboptimatility gap & time horizon, we can get logarthmic regret.

Limitations:
1. Dependence of suboptimality gap and time horizon in tuning the param -m.

# Parameters
<!-- $$
